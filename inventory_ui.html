Database Discovery

Cross-account EC2 inventory: MySQL, PostgreSQL, MongoDB — filtered by region and spoke account. Data is the last discovery write to S3; use Reload after a new scan.

How this page works
  1. Region — only EC2 rows discovered in that region.
  2. Account — dropdown lists accounts that have data in the selected region (not the whole org).
  3. Empty DB column — probe ran; no supported engine found on that host.
